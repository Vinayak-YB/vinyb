## Work

### ApraPipes {#aprapipes}
An open-source image and video processing framework in C++ and Python. Contributed to JPEG/H264 encode-decode modules, MP4 read/write, thread and memory management, and GPU-accelerated image processing.

[GitHub](https://github.com/apralabs/ApraPipes)

### ApraNVR {#apranvr}
A network video recorder built on ApraPipes with OpenGL and GTK. Designed the pipeline architecture and wrote most of the backend. Integrated with Nvidia Jetson devices (AGX, Orin, NX).

### SIP Intercom {#sip-intercom}
Intercom application on Baresip SIP library with Redis, deployed on Raspberry Pi and Broadcom boards. Worked on ALSA audio, SIP protocols, and implemented a Whisper-based voice-to-text module in ApraPipes.
